## Pricing Notes

- Cheapest property (Terracotta group): KES 600 — roughly 4% of starting cash, affordable within the first lap for most players.
- Most expensive property (Karen): KES 4,000 — roughly 27% of starting cash, in line with the standard Monopoly ratio of top-tier property to starting cash.
- All four Reserves are priced identically (KES 2,000), matching the standard railway-pricing convention where all stations in a set cost the same regardless of position.
- Utilities priced below Reserves (KES 1,500) since their rent model scales with dice roll rather than a fixed/group-based multiplier — kept lower to reflect their higher variance/lower reliability as an investment.
- Income Tax (position 4, KES 1,500) is set higher than Luxury/Super Tax (position 38, KES 1,000) — intentionally inverted from some Monopoly editions, but kept consistent with the original draft's ordering; flag if you'd prefer these swapped to match classic Monopoly convention (Luxury Tax traditionally the smaller, fixed late-game hit).
